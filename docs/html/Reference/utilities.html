GoH.utilities¶

The utilities package provides a collection of functions used to prepare data for different analysis tasks. These are largely helper functions, but some can operate on their own.

Examples

>>> text = GoH.utilities.readfile( input_dir, filename)
>>> spelling_dictionary = GoH.utilities.create_spelling_dictionary( wordlists, directory )

GoH.utilities.rate_per_doc(corpus_statistics)[source]¶

Gather a list of doc_id and error_rates.

References

Uses the data generated by GoH.reports.process_directory().

Note

Use GoH.reports.process_directory() before running stats_to_df

Parameters:corpus_statistics (dict) – List of dictionaries with the information about all of the files in a directory.
:param The corpus data should be formatted as follows:::
{
‘doc_id’: document, ‘num_tokens’: len(tokens), ‘num_unique_tokens’: len(set(tokens)), ‘num_errors’: error_total, ‘error_rate’: rate, ‘errors’: error_report

}

}
